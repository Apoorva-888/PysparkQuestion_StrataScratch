This repository contains **PySpark solutions** to coding questions from **StrataScratch**.  
Each question is stored in its own folder with a `solution.py` file and a short README describing the problem.

---

## How to Run

1. Open the folder of the problem you want to run.
2. Open `solution.py`.
3. Run the code in your PySpark environment.
4. Check the output using `.show()` or `.toPandas()`.

---

## Notes

- All solutions use **PySpark DataFrame API**.  
- Output column names match the expected StrataScratch format.
